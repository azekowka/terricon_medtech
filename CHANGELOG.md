# Changelog — MedServicePrice.kz

Все значимые изменения проекта.

## [Unreleased]

## [0.4.0] — Каталог врачей (клон вкладки idoctor /doctors)

### Added
- **Скрейпер idoctor.kz** (`scripts/scrape_idoctor.py` + `idoctor_extract.py`):
  извлекает встроенный в страницы JSON `initialDoctorsList` (Next.js RSC) по всем
  **18 регионам Казахстана** и специализациям. Собрано **~1500 врачей** с полными
  данными: ФИО, специализации, категория, стаж, рейтинг, отзывы, клиники с
  **графиком работы по дням**, адресом, гео-координатами и ценами приёма; плюс
  «Лечение заболеваний» с профиля. Вежливый (задержки, ретраи).
- **Backend**: таблица `doctors`, загрузчик (`doctors_loader.py`), API
  (`routers/doctors.py`): `/api/doctors/meta` (регионы+специализации), `/api/doctors`
  (фильтры: регион, специализация, цена, рейтинг, стаж, дети, онлайн-запись,
  сортировка, пагинация), `/api/doctors/{id}`.
- **Frontend `/doctors`** — клон каталога врачей: попап выбора города (18 регионов),
  сайдбар фильтров (специализация с поиском+счётчиками, цена, рейтинг, стаж,
  переключатели), карточки врачей (фото, специализации, стаж/категория, приём
  детей, рейтинг+отзывы, клиника+адрес, цена со скидкой, онлайн-запись, «Записаться»),
  пагинация, модалка врача (клиники, график, заболевания, запись).
- Пункт «Врачи» в навигации.

## [0.3.0] — Реальные прайсы клиник

### Added
- Источник **`real`** (`backend/app/parsers/real_extract.py` + `real_prices.py`):
  парсинг настоящих прайс-листов 7 клиник из `data/real_prices/` в форматах
  **PDF / DOCX / XLSX / XLS** (UMC, ННМЦ, Green Clinic и др.) — ~1450 реальных цен.
- Колоночное распознавание таблиц (столбцы «Наименование» + цена для граждан РК) и
  построчный разбор PDF со срезанием тарифного кода.
- Поддержка `.xls` (xlrd). Тесты на real-экстракцию всех форматов.

### Fixed
- Непривязанные услуги показывали имя отклонённой fuzzy-подсказки из справочника —
  теперь отображается настоящее (raw) название услуги.

### Stats
- 29 клиник · 3048 цен · 17 источников · 6 городов.

## [0.2.0] — Официальный справочник услуг

### Changed
- Словарь услуг заменён на **официальный справочник из ТЗ**
  (`data/source/services_reference.xlsx`) → **1230 услуг, 122 специальности**.
  Категория выводится из префикса тарифного кода (A/B/C/D), курируемые синонимы/цены
  накладываются через fuzzy-match. Поля `specialty` и `tarif_code` показаны в UI.

### Fixed
- `/api/services` обрезал список до 500 → на главной «Процедура» = 0 услуг.
  Добавлены счётчики по категориям в `/api/meta`, сортировка `?sort=popular`.

## [0.1.0] — MVP

### Added
- Backend (FastAPI + SQLAlchemy): модели, поиск/автодополнение, нормализация
  (exact + rapidfuzz), дедупликация, raw-слой, очередь ручной разметки, история цен,
  подписки, журнал парсинга, APScheduler.
- Парсеры: live web-скрейперы (kdl/invitro/helix), seed-источник, `fixtures`
  (PDF/XLSX/DOCX), мультиформатный экстрактор.
- Frontend (Next.js 14 + TS + Tailwind): поиск с фильтрами/сортировкой, сравнение,
  карта (Leaflet), история цен (Recharts), карточки клиник, админка, адаптивность.
- Docker/docker-compose, README, ARCHITECTURE, презентация (7 слайдов), 9 тестов.
- Прошёл adversarial multi-agent review: исправлено 17 подтверждённых замечаний.
